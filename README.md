# WME-Open-Other-Maps
Open various other maps at the current WME location (if supported).

Currently supported:
* Google Maps
* Mapillary
* Terraserver
* Wikimapia
* Bing Maps
* Open Street Map
* Yandex
* Here

Now also lets you open Waze from other maps!  Sorry - no way to disable this as the settings cannot be access across websites.
